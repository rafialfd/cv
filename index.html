Mengembangkan aplikasi web interaktif untuk menghitung poin FIFA berdasarkan hasil pertandingan antar negara, dengan logika perhitungan dinamis menggunakan JavaScript serta tampilan responsif berbasis HTML dan CSS.
Kantin Bundo Website
Membangun website bisnis menggunakan WordPress untuk mendukung operasional dan branding kantin, termasuk pengelolaan konten, tampilan UI, serta struktur informasi yang mudah diakses pengguna.
2025 Spotify Wrapped Analytics
Melakukan analisis data pemutaran musik pribadi menggunakan Power BI, dengan visualisasi interaktif untuk mengeksplorasi pola kebiasaan mendengar, tren genre, serta insight dari histori data sepanjang tahun 2025.

Mengembangkan game berbasis web untuk menavigasi artikel Wikipedia secara efisien, dilengkapi sistem autentikasi (JWT), backend Node.js, dan database PostgreSQL untuk menyimpan serta menganalisis performa pengguna seperti jumlah klik dan waktu penyelesaian.
